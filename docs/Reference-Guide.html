3 Reference Guide

The Guile GI project provides two Guile modules, (gi introspection), and (gi object). The (gi introspection) module is an interface for dynamically creating Guile modules for GObject-based libraries by examining the information in their .typelib files. The (gi object) module is an interface for creating new GObject classes and instances in Guile. These two modules allow one to write programs using GObject-based C libraries like GTK3 or WebKitGTK.
