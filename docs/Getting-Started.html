1 Getting Started

To get things started, we will try to run a very simple GTK+ based GUI application using the bindings provided by Guile GI.

First, create a (Gtk) Guile module. 

 guile -L . -c '(use-modules (gi)) (display (export-typelib "Gtk" "3.0"))' > Gtk.scm

Then, create a small Guile script called hello.scm with the following content and save it somewhere (FIXME, this doesn’t work): 

(use-modules (gi)
             (Gtk))
(define window (Window #:title "Hello World"))
(show window)
(connect window main-quit)
(main)

Before we can run the example application, we need to install Guile GI, GTK+, and their dependencioes.

When it is run, it will create an empty window with a title. Are you impressed?
